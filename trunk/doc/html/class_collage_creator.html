Detailed Description

This widget allows to create a phtoto collage.

The widget allows to create a photo collage based on parameters defined by the user such as the collage shape, the space between each photo in the collage, etc.

CollageCreator is integrated in the QuteCollage's main window, but can also be used as a standalone program.
